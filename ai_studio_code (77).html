FAE Application Insights

The GD25CIZ120C7S is a Converter-Inverter-Brake (CIB) module, representing the highest level of power stage integration. It includes the 3-phase input rectifier, the 3-phase output inverter, and the braking chopper IGBT. This module is the fastest way to design a complete, compact motor drive up to 7.5kW, drastically reducing design time and manufacturing complexity.
